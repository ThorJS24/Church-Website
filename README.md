# Salem Primitive Baptist Church Website

A modern, divine church website built with Next.js 14, Firebase Auth, and MongoDB.

## ✨ Features

- **Heavenly UI/UX** - Divine light and dark themes
- **Firebase Authentication** - Google OAuth integration
- **MongoDB Database** - User profiles and data storage
- **Responsive Design** - Mobile-first approach
- **Modern Stack** - Next.js 14, TypeScript, Tailwind CSS

## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd church-website-nextjs
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   Copy `.env.example` to `.env.local` and fill in your values

4. **Run development server**
   ```bash
   npm run dev
   ```

## 🌐 Deployment

This app is optimized for Railway deployment with automatic builds and MongoDB integration.

## 🎨 Theme

- **Dark Theme**: Mystical, sacred, elegant atmosphere
- **Smooth Transitions**: 0.6s cubic-bezier animations

- User authentication and profiles
- Church information and contact
